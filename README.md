## Descrição
Repositório encurtador de URLs

## Iniciar com Docker
```bash
$ docker compose up -d
```

## Instalar dependências
```bash
$ npm install
```

## Compilar e Iniciar o projeto
```bash
$ npm run start

# watch mode
$ npm run start:dev
```

## Iniciar Testes
```bash
# Testes Unitários com cobertura
$ npm run test
```

## Acessar Documentação
```bash
# Após iniciada a aplicação, acessar:
http://localhost:3000/api
```
